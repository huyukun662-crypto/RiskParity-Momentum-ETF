# v5d_final 策略描述

> A 股跨资产 ETF：风险平价 × 动量共振 × 过热高低切 × 大盘趋势择时

---

## 一、一句话定位

**在 64 只 A 股跨资产 ETF（宽基 + 行业 + 风格 + 跨境 QDII + 商品 + 债券）中，每周五按"风险平价筛选 + 动量共振评分"挑 3 只建仓；每日监控过热切换；当沪深 300 过去 120 日累计收益 < -8% 时，全仓切黄金避险。**

## 二、策略哲学：四层叠加

传统动量策略的两大死穴：
1. **系统性下跌时，所有动量标的同向下跌**（momentum crash）
2. **趋势持续时容易在冲顶附近加仓**（RSI/logbias 已过热）

v5d 用四层依次解决：

| 层级 | 要解决的问题 | 技术方案 | 贡献 |
|---|---|---|---|
| ① **风险平价过滤** | 避开死水（σ 太低）和狂风（σ 太高） | 年化波动率 ∈ [8%, 28%] + 相关性最低 5 只 | 低相关候选池 |
| ② **动量共振评分** | 仅选有强趋势 × 稳定的标的 | `score = (exp(β·252) − 1) × R²` | 趋势 Alpha |
| ③ **过热高低切** | 避免冲顶追涨 | logbias > 类别阈值 OR (RSI14 > 78 AND RSI 下行) | 降低个股回撤 |
| ④ **大盘趋势择时** ⭐ | 躲避系统性熊市 | HS300 120 日收益 < -8% → 全仓黄金 | 降低系统性回撤 |

**④ 是 v5d 的关键设计** —— 把纯动量多头升级为趋势跟踪型策略，OOS 段 Sharpe 2.12 的主要贡献来源。

## 四、关键指标详解

### 4.1 动量共振评分

对 **过去 20 个交易日的对数价格** 做一元线性回归：

```
ln(Pₜ) = α + β · t + ε
```

**momentum_score** = **年化漂移** × **趋势强度**：

```
score = (exp(β × 252) − 1) × R²
```

- `(exp(β·252)-1)`：把日对数斜率 β 年化为可比收益率
- `R²`：回归决定系数 ∈ [0, 1]，衡量"价格沿直线上升"的稳定性

**为何叫"共振"**：一只标的必须**既涨得快 (β 大)** **又涨得稳 (R² 高)** 才能拿到高分。单纯暴涨暴跌（低 R²）不会被选中。

### 4.2 logbias 指标（冲顶程度）

```
EMA30(ln P) = 30 日指数移动均线
logbias     = (ln Pₜ − EMA30(ln P)) × 100
```

物理意义：**当前价相对 30 日均线的对数百分比偏离**。
- logbias = 10 → 当前价比 30 日均线高约 10%
- 经验阈值（按类别不同）：
    - 股票类 16.5（涨太高容易回调）
    - 商品类 11.0（商品更易均值回归）
    - 红利类 6.0（低波品种，5% 偏离即过热）
    - 债券类 ∞（不判过热）

### 4.3 RSI14（Wilder 平滑）

```
gain = max(Pₜ − Pₜ₋₁, 0)
loss = max(Pₜ₋₁ − Pₜ, 0)
avg_gain = Wilder EMA (α = 1/14) of gain
avg_loss = Wilder EMA (α = 1/14) of loss
RS       = avg_gain / avg_loss
RSI      = 100 − 100 / (1 + RS)
```

**过热判定**：RSI > 78 **且** RSI 在下行（rsi_diff < 0） → 说明已冲高回落。

### 4.4 Regime 滤波

```
ret_120d = P_HS300(today) / P_HS300(today − 120 TD) − 1

if ret_120d < -8%:
    → 全仓 518880 黄金ETF（防御）
else:
    → 正常进攻选股
```

**120 日参数选择依据**（由 IS 2018-2023 扫参决定）：
- lookback ∈ {60, 90, 120} 中 120 在所有维度都最优
- 短窗口（60 日）对震荡期过度敏感 → 频繁误触发
- 长窗口（120 日）要求**持续性下跌**才防御 → 过滤噪声

**为何选黄金而非国债作防御资产**：
- IS 扫描 36 组，Top 10 全部是黄金防御
- A 股熊市期（2018/2022）黄金独立于 A 股走出牛市，而国债仅温和上涨
- 2023 后贵金属 supercycle 更强化这一结构性 Alpha

## 七、回测关键事件（Full 2020-2026.4）

### 7.1 Regime 防御触发分布（共 52 次）

| 时段 | 市场情境 | 防御次数 | 效果 |
|---|---|---|---|
| 2020-04 ~ 2020-06 | 疫情后段修复（120D 仍记疫情跌） | ~8 次 | 部分错过反弹，代价约 +1pp |
| 2022-04 ~ 2022-11 | A 股连续阴跌 | ~24 次 | **成功避开 -15% 级跌幅** |
| 2023-09 ~ 2024-02 | 二次探底 | ~15 次 | **OOS 段核心贡献**（Sharpe 2.12 来源） |
| 2024-08 ~ 2024-09 | 政策刺激前夜 | ~5 次 | 部分错过 924 反弹 |

### 7.2 最大回撤 -13.22%（2020-02-13 → 03-17）

**原因**：120 日 lookback 对 V 型急跌反应不够快。2020-01 疫情爆发时，HS300 过去 120 日（含 2019Q4 牛市）仍为正收益 → 不触发防御 → 吃满疫情下跌。

**这是 regime 设计的已知 trade-off**：
- 优点：长窗口过滤噪声，减少误触发
- 缺点：V 型急跌反应慢

替代方案（可选但 IS 未胜出）：双窗口 regime（60 + 120）任一触发即防御 → IS Sharpe 下降 0.1，未采纳。

### 7.3 OOS 2024-2026.4 表现极佳

OOS Sharpe **2.12** vs IS 1.60 → OOS/IS = 1.33

- 2024-01 年初崩盘：regime 提前触发 → 完美避开 -10% 跌幅
- 2024-10 924 反弹后：快速切回进攻，吃到创业板 + 中证 1000 动量
- 2025 商品牛市：黄金 / 豆粕 / 有色多次轮入
- 2026-03 回撤：因 n=3 分散，仅回撤 -5% 级

## 九、已知局限与改进方向

### 局限

1. **V 型急跌保护有限**（120 日 lookback 延迟）
2. **换手偏高**：周度 234 次 + 日度 10 次 = 6.3 年共 244 次调仓，年化换手 ~38 次
3. **数据依赖**：需要 64 只 ETF 全量历史 + 沪深 300 基准
4. **参数敏感性**：lookback 从 120 → 60 会使 Full Sharpe 下降约 0.3（但 Top 3 参数都在 Sharpe 1.87±0.1 范围内，非尖锐）
5. **池子依赖跨境 QDII**：剔除跨境后 Full Sharpe 降至 1.26 左右

### 改进方向（留给后续）

- **双 regime 组合**：60 日 + 120 日并联，前者抓急跌、后者过滤噪声
- **动态仓位**：bull 时超配、neutral 时减半、bear 时防御
- **部分防御**：bear 阶段保留 30% 现金 + 70% 黄金，提升切回时机灵活性
- **跨品种黄金替代**：白银 / 原油 / 金矿股 ETF 作防御备选
- **尾部风险对冲**：认沽期权（若有 ETF 期权）进一步降尾部
